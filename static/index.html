Executive Summary

This document specifies the software verification and validation test methods and results for the Smart Evaluation App.

  1. Introduction

    1. Overview

      Testing will be conducted to determine that the software meets its requirements as specified in the following documents:
      Software Requirements Specification

    2. Test Platform

      Unless specified otherwise by the individual test method, all tests are performed on a compatible Android SmartPhone and iPhone.

    3. Test Level

      Software Item - this test level verifies the correct operation of each software unit contained within the software item. No additional system hardware is required to complete this level of verification.
      External Software Interface - this test level verifies the correct operation of each external software interface. The system under test will be verified against an applicable external interface device as specified in the test method.
      System - this test level verifies the correct operation of the software through the entire system. System level activities such as event sequence tests are performed in a controlled environment by system integrators.
